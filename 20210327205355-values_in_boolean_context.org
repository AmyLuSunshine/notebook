#+title: Values in Boolean Context
#+ROAM_TAGS: Javascript

- tags:: [[file:20210327205115-javascript.org][Javascript]]

* number

** false: 0

** other numbers are true

* string

** false: empty string

** other strings are true

* undefined

** false

* null

** false

* NaN

** false

* Examples

  Following code all prints 1.

#+begin_src js

console.log(0 || 1);

console.log("" || 1);

console.log(NaN || 1);

console.log(undefined || 1);

console.log(null || 1);
#+end_src
