#+title: Binding Keys to Commands
#+ROAM_TAGS: Emacs

- tags :: [[file:20201028193935-emacs.org][Emacs]]

* Example: Binding "s-s" to my-lookup-cam 
#+begin_src elisp
(global-set-key (kbd "s-s") 'my-lookup-cam)
#+end_src

* Function keys
#+begin_src elisp
(global-set-key [f1] 'my-lookup-cam)
#+end_src
