#+title: Dynamic Host Configuration Protocol (DHCP)
#+filetags: :Networking:

- tags :: [[id:e3c4ce8a-faa5-4e54-b368-03a0dd8ead33][Networking]]

DHCP automatically assigns:

- IP address
- Subnet mask
- Default gateway
- DNS Server's IP address
- other Internet configuration parameters
