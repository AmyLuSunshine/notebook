* Usage

  #+begin_src sh
awk 'PATTERN { ACTION }' file
  #+end_src

* Built-in variables
  
** NF
   number of fields
  #+begin_src sh
awk '{print NF}' file
  #+end_src

** NR
   number of rows   
  #+begin_src sh
awk '{print NR}' file
  #+end_src   

* Formating out put
#+begin_src sh
awk '{printf()}' file
#+end_src
